## Prerequisites

Before contributing, ensure you have the required tools.  
**Tip**: Use **[GitHub Codespaces](#one-click-start-no-setup-needed)** to skip local setup entirely.

### Installing an IDE

We recommend **VS Code** — free, lightweight, and supports all languages in this repo.

- Download: [code.visualstudio.com](https://code.visualstudio.com/)
- Install extensions via **Extensions tab** (`Ctrl+Shift+X`):
  - Search for your language (e.g., "Python", "Rust", "Go") and install official extensions.

---

### Installing Git

Git is required for cloning and contributing.

| OS | Command |
|----|--------|
| **Linux (Ubuntu/Debian)** | `sudo apt install git` |
| **Linux (Arch)** | `sudo pacman -S git` |
| **macOS** | `brew install git` (via [Homebrew](https://brew.sh)) |
| **Windows** | [gitforwindows.org](https://gitforwindows.org) |

Verify: `git --version`

---

### Installing Node.js and NPM

Many frontend and tooling tasks require Node.js.

- Download: [nodejs.org](https://nodejs.org) (LTS version)
- Or via package manager:
  - macOS: `brew install node`
  - Linux (Ubuntu): `sudo apt install nodejs npm`
  - Arch: `sudo pacman -S nodejs npm`

Verify: `node --version && npm --version`

> **Note**: The following depend on Node.js:  
> JavaScript, TypeScript, React, Vue, Svelte, Tailwind, SASS, etc.

Most languages have **official installers** or **package managers**. Below are quick commands.

## How to Contribute (3 Steps)

1. **Pick an open issue**  
   → Look for `good first issue` + your favorite language

2. **Write "Hello World"**  
   → Create a file in `src/[language]/`  
   → Example: `src/python/hello.py`

3. **Open a Pull Request**  
   → GitHub guides you  
   → We’ll merge it in < 24 hours

That’s it. You just contributed to open source.
